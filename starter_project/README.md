# 🚀 Starter Project — Build Your Own!

This is your blank canvas. Use what you learned in the exercises to build
something from scratch with GitHub Copilot.

## Project Ideas

Pick one, or come up with your own:

- **Student Grade Tracker** — Track courses, assignments, and calculate GPA
- **Weather Dashboard** — Fetch weather data from an API and display it
- **URL Shortener** — Simple link shortener with redirect tracking
- **Habit Tracker** — Daily habit tracking with streak counts
- **Quiz Generator** — Generate quizzes from a topic using Copilot

## How to Start

### 1. Define Your Project

Open Copilot Chat and describe your project:

> "I want to build a {project name}. It should {key features}. Help me plan
> the structure and create the initial files."

### 2. Use the Spec-Driven Approach

1. Edit `.specify/spec.md` with your feature description
2. Ask Copilot to generate a plan from the spec
3. Break the plan into tasks

### 3. Build Iteratively

1. Start with `main.py` — build the core feature first
2. Add utilities in `utils.py`
3. Write tests in `test_main.py` as you go
4. Expand from there!

## File Structure

```
starter_project/
├── README.md          ← You are here
├── main.py            ← Your main application
├── utils.py           ← Helper functions
└── test_main.py       ← Tests for your code
```

## Run Your Project

```bash
python starter_project/main.py
```

## Run Your Tests

```bash
pytest starter_project/test_main.py -v
```
